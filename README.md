# 🚀 AI Career Copilot

AI Career Copilot is an AI-powered resume analysis platform that helps job seekers evaluate their resumes, identify skill gaps, generate personalized learning roadmaps, and prepare for interviews.

## 🌐 Live Demo

https://ai-career-copilot-0p2d.onrender.com

---

## ✨ Features

* 📄 Resume Upload (PDF & DOCX)
* 🎯 Role-Based Resume Analysis
* 🛠 Skill Gap Identification
* 🗺 Personalized Learning Roadmap
* 🎤 AI-Generated Interview Questions
* 📚 Analysis History Tracking
* 🔐 Secure Authentication System
* 🤖 AI-Powered Career Guidance
